## 实现一个计算器

### 要求

给定一个 **字符串表达式**，实现 **整数四则运算**。

下面是字符串表达式的样例：
```
1+1
1-1000
999*1
9/2
```

注意：

- 操作数和结果都是 32 位整数
- 除法是整除，且不需要考虑除数为 0 的情况


### 代码编写

在 `Calculator` 文件中，你 **只需要** 将代码写在两个`===== YOUR CODE HERE =====` 之间的区域，其他代码禁止修改。

请将字符串 `s` 作为表达式进行处理，并把算出来的结果放在整数 `result` 中。
